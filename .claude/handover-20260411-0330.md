## サブエージェントへの指示で学んだこと

### 必ず渡すもの
1. **品質チェックリスト全文**（`.claude/knowledge-spec-quality-checklist.md`）
   - 初期のエージェントに渡し忘れて品質にばらつきが出た
2. **ソースコードのパス**（対象画面のAngular HTML + TypeScript）
3. **旧specのアーカイブパス**（セレクター参考用）
4. **テスト実行コマンド**（`source .env && AGENT_NUM=1 SKIP_GLOBAL_SETUP=1 bash run-test.sh ...`）

### よくあるfail原因
1. **beforeAllのログイン**: 新コンテキストがリダイレクトされてloginページに行かない → `clearCookies` + `localStorage.clear` + `/login`チェック必須
2. **ALLテストテーブルの操作が遅い**: 102フィールドの描画/送信に時間がかかる → タイムアウト延長
3. **必須フィールド**: メール欄が必須 → ダミー値入力が必要
4. **CSVダウンロード**: モーダルなしで直接DLが始まる場合がある
